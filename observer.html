Observer layer

Cardinal produces a world. It also needs to be watchable. The observer layer doesn't affect the simulation. It watches it.

Observation modes

Raw log stream

Every tick, every agent, every action. For analysis.

Event feed

Significant events only, in readable form. For monitoring.

Chronicle summary

Compressed narrative from the chronicle tick. Readable as history.

Spatial view

2D map render with agent positions, territories, resource overlays, event markers.

Agent inspector

Full internal state, memory contents, relationship map, current need levels.

Lineage graph

Generational tree. Trait drift visualization across generations.

Knowledge map

Track a knowledge item across agents and time. See it spread, mutate, or die.

Civilization without design. Culture without instruction. History without author. Meaning without assignment.

And seeing what survives anyway.
