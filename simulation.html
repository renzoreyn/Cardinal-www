Simulation loop

The world only appends history. Nothing gets edited out.

Ticks

Micro tick

Every frame

Agent movement and physics. Resource consumption and regeneration. Environmental state update. Collision and proximity detection.

Cognitive tick

Every N micro ticks

Need state evaluation. Action selection via policy. Memory read/write. Social evaluation. Communication signal emission and reception.

Chronicle tick

Every simulated day

Event aggregation and tagging. Memory decay application. History compression. Long-term pattern detection. Cultural marker evaluation.

Chronicle tick reads the event stream and builds compressed narrative: not "agent 0042 gathered berries at tick 18273" but "group near the river valley has sustained itself on river resources for 14 simulated years."
